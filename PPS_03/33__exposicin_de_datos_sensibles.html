3.3. - Exposición de datos sensibles.

Una de las grandes preocupaciones tanto de usuarios como de las organizaciones, es la seguridad y privacidad de los datos sensibles: como por ejemplo información de carácter personal, laboral, bancario, financiero o de salud. Este tipo de datos sensibles pueden ser robados, modificados o vendidos por los cibercriminales.

La primera consideración a tener en cuenta para administrar datos sensibles y evitar su exposición, es la correcta clasificación de la información en datos sensibles y datos no sensibles.  A los datos sensibles, se les deben aplicar medidas de seguridad adicionales: evitar que estos sean almacenados y protegerlos métodos de cifrado robustos, para evitar que estos puedan ser interceptados en las comunicaciones. Se recomienda emplear protocolos seguros como TLS (Transport Layer Security) y PFS (Perfect Forward Secrecy).

Para evitar ataques por este riesgo se recomienda adoptar, entre otras, las siguientes medidas de prevención:

  • Evitar el almacenamiento de datos sensibles. En caso de que el almacenamiento de esta información sea indispensable, estos datos deberán guardarse cifrados, con  algoritmos de cifrado fuertes y seguros.
  • Las contraseñas débiles son brechas de seguridad fáciles de explotar. Además del empleo de algoritmos de cifrado fuertes es muy recomendable utilizar contraseñas seguras. Se recomienda modificar sus contraseñas con regularidad y mantener una contraseña única para cada plataforma diferente.
  • El mantenimiento de copias de seguridad seguras y protegidas, es de especial utilidad para  mitigar las pérdidas de información, ocasionadas como consecuencia del robo o alteración malintencionada de los datos.
  • Es importante que solo se utilicen puertas de enlace de autenticación seguras. Con la utilización de seguridad estándar avanzada, como SSL y TSL, puede asegurarse de que la información que fluye entre un navegador web y un servidor web no solo esté cifrada, sino que también permanezca privada. Además, HTTPS ofrece un protocolo de comunicación seguro para las aplicaciones que lo utilizan.
