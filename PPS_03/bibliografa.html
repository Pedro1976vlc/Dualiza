Bibliografía

Fuentes documentales utilizadas en esta unidad

  • Proyecto OWASP Top Ten para aplicaciones web: Listado de los riesgos más destacados OWASP.
  • Proyecto OWASP Top Ten para aplicaciones móviles: Listado de los riesgos más destacados OWASP.
  •  Manual del estándar de verificación de aplicaciones seguras (ASVS): Documento de referencia con los niveles y requerimientos que las aplicaciones han de cumplir para ser certificadas.
  • Manual sobre pruebas para aplicaciones web (WSTG): Guía que describe los principales problemas de las aplicaciones web con información técnica, soluciones, pruebas y herramientas relacionadas.
  • Guía de  pruebas para aplicaciones móviles (MSTG): Proyecto que incluye una guía similar a la de aplicaciones web junto con una adaptación del estándar ASVS para móviles (MASVS) y una hoja de cálculo para realizar la evaluación.
  • Canal de la fundación OWASP Youtube:  Canal oficial de la fundación, contiene información variada como conferencias, charlas y manuales
