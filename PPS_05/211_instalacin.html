2.1.1.- Instalación.

La instalación de git es sencilla, tanto en Windows, Mac como en Linux, por ejemplo en Debian/Ubuntu simplemente ejecutar el comando:

$ apt-get install git

En Mac se encuentra integrado en las herramientas de Xcode, también se puede instalar la aplicación proporcionada por GitHub.

Por último en Windows existen diferentes versiones se puede descargar desde http://git-scm.com/download/win o utilizar la herramientas proporcionada por GitHub, que además de un entorno gráfico incluye los comandos desde el terminal.

Los comandos y forma de trabajar con Git es independiente del sistema operativo utilizado.

Para comprobar que se ha instalador simplemente ejecutar el comando git en un terminal.
